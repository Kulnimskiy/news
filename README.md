## 1. Задание

Model - Новости
Заголовок - varchar
Содержимое - text
pub_date - дата и время публикации

Views:
Создать новость
Редактировать новость
Удалить новость
Показать списком отфильтровав по дате
Посмотреть детально

Все обернуть докером, приложение должно использовать Django и Postgres

### 2. Инициализация сервера и бд

make init
